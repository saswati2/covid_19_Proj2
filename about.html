Covid-19(Till 07/22/2020)

About the Visualization

Overview

Started back in Dec, 2019, COVID-19 has spread out each and every countries around the world. The Martini Glass visualization structure is used to tell the story for the impact as on 07/22/2020.

The D3 JavaScript library in combination with HTML and CSS are utilized to create the visualization. The publicly available data has been downloaded and referenced from HTML and JavaScript files. The data visualization components in this content are animated bar-chart, line graphs with multiple lines and scatterplot which have been created using D3.js.

This narrative visualization has been created to find out following (utilizing data available till 07/22/2020):

1. Top 50 COVID-19 infected countries

2. Growing trend of total number of Confirmed cases of COVID-19 for top 5 infected countries

3. Growing trend of total number of Deaths due to COVID-19 for top 5 infected countries

4. Correlation between Total number of confirmed cases, recovered cases, deaths due to COVID-19 and Population, Income, Region of the different countries around the world.

Narrative Visualization

This Martini Glass visualization structure begins with an author-driven approach, initially using following scenes:

Home Page: This first page sets the context and provides the pictorial representation of total number of confirmed cases for top 50 infected countries.

Confirmed Cases Page: This is the 2nd scene which provides visual animated representation of growing trend of COVID-19 confirmed cases for top 5 most infected countries namely US, Brazil, India, Russia and South Africa. The annotation has been used to clearly mark the trend pattern. The country specific color scheme has been used to aid the users to understand the chart better. Following observations can be easily inferred:

1. US is in already 2nd wave as the line curve representing the total number of confirmed cases was having flattening trend around mid-May, 2020 and it is again gaining its momentum since mid-June, 2020.

2. Brazil, India and South Africa have not reached to its first peak yet as the line curves representing the total number of confirmed cases are showing continuous upward trend with no sign of flattening of the curve.

3. The line curve representing total number of confirmed cases for Russia is showing the sign of flattening the curve. It is still in first wave though.

Deaths Page: This is 3rd scene which provides the trend of total deaths of COVID-19 in those top 5 infected countries since its inception. Same annotation style and country specific color scheme have been used to aid the users. Following observations can ben easily interred through this page:

1. Second Wave of Deaths has been started in US around mid-June 2020.

2. Countries (Brazil, India and South Africa) which are in first wave in terms of total number of confirmed cases (as depicted in scene 2), are in first wave in terms of total number of Death as well. Russia has shown the sign of flattening the curve for the first wave in terms of total number of deaths.

Final Page: Once the authorâ€™s intended narrative (guided journey) is complete to find out Top 50 infected countries, growing trend of total number of confirmed cases and deaths for top 5 infected countries, the visualization opens to a reader-driven stage where the user is free to interactively explore the data. The Final Page is a chance for the user to explore the correlation between Total number of confirmed cases, recovered cases, deaths and Population, Income, Region of the countries.

Preparation

The data is cleaned and organized beforehand in MS Excel so that the complete focus remain in visualization. Following steps have been taken:

 1. Metadata_Country.csv: Population from API_SP.POP.TOTL_DS2_en_csv_v2_1217749.csv (renamed to pop_country.csv) has been added (Country code is used for excel VLOOKUP)

 2. time_series_covid19_confirmed.csv: Original timeseries data for COVID-19 confirmed cases for all the countries

 3. time_series_covid19_confirmed - Top5 - transposed.csv: Transposed the time series columns into rows for top five countries

 4. time_series_covid19_deaths.csv: Original timeseries data for COVID-19 deaths for all the countries

 5. time_series_covid19_deaths - Top5 - transposed.csv: Transposed the time series columns into rows for top five countries

  6. time_series_covid19_recovered.csv: Original timeseries data for COVID-19 recovered cases for all the countries

 7. latest_covid19_confirmed_deaths_recovered.csv: 07/22/2019 country wise data for all the countries (Country name is used for excel VLOOKUP)
