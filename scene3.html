Covid-19(Till 07/22/2020)

While number of confirmed cases are growing, the death toll is also increasing day by day.

This line graph helps to find out the pattern of growing number of Covid 19 deaths for US, Brazil, India, Russia and South Africa which have most number of confirmed cases.

It has five lines one for each country. Color legends are added for visual affordance. This uses same annotation style from previous page is used.

Following observations can be easily inferred:

1. Second Wave of Deaths has been started in US around mid-June 2020

2. Countries (Brazil, India and South Africa) which are in first wave in terms of total number of confirmed cases (as depicted in scene 2), are in first wave in terms of total number of Death as well. Russia has shown the sign of flattening the curve for the first wave in terms of total number of deaths.

3. The line curve representing total number of confirmed cases for Russia is showing the sign of flattening the curve. It is still in first wave though.
