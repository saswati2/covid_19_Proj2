Covid-19(Till 07/22/2020)

This scatter plot provides the opportunity to drill down the data (as of 07/22/2020) further through user interaction. It provides the user a chance for the user to explore the correlation between Total number of confirmed cases, recovered cases, deaths and Population, Income, Region of the countries.

Menu and selection parameter is provided to access the impact on each country. The transition happens in an animated way.

Tooltip is provided to display further details for each plotting.
